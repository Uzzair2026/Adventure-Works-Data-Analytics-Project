# Adventure-Works-Data-Analytics-Project
Data Analytics project using Excel, Power BI, Tableau, and MySQL on Adventure Works dataset.

**Power BI | Adventure Works Sales Dashboard | Data Visualization | KPI Reporting | Business Intelligence | DAX | Power Query**

## 📊 Adventure Works Sales Dashboard

This project presents an interactive **Power BI dashboard** built using the Adventure Works dataset. It provides a comprehensive analysis of sales performance, customer activity, profit, and regional trends to support business decision-making.

## 🔑 Key Features

* Sales, Orders, Customers & Profit KPIs
* Product-wise Sales Analysis
* Customer Order & Profit Analysis
* Regional Performance Dashboard
* Sales by Region Visualization
* Interactive Slicers (Year, Quarter, Sales Territory)

## 🛠 Tools & Technologies

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization

## 📈 Business Insights

* Monitors overall business performance through KPI cards.
* Identifies top-performing products and regions.
* Analyzes customer contribution to revenue and profit.
* Tracks regional sales trends for strategic decision-making.
